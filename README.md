# k2dasm

k0dasm is a disassembler for NEC 78K/II uPD78213 used within Brother KH270 Control Box.

This work is derived from https://github.com/mnaberez/k0dasm

## Usage

    python3 dasm.py -f kh970-rom-41869ABCD.bin > trace.log
